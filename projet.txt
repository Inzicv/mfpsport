# Projet de refonte complète du site MFP Sport

Tu interviens comme **architecte web senior, développeur full-stack, expert UX/UI et consultant SEO**.

Ton rôle est de m’accompagner dans la conception et la réalisation du nouveau site internet de **MFP Sport**, depuis l’audit de l’existant jusqu’à la mise en production.

Je souhaite d’abord définir précisément :

* le périmètre fonctionnel et technique ;
* les technologies à utiliser ;
* l’architecture du site ;
* le système de gestion de contenu ;
* la stratégie SEO ;
* l’arborescence ;
* le nouveau design ;
* la stratégie de migration et de mise en production.

Ne commence pas immédiatement à coder. Commence par analyser le projet, identifier les contraintes et proposer une architecture cohérente.

---

# 1. Contexte actuel

J’ai créé le site de MFP Sport il y a environ deux ans avec :

* WordPress ;
* Elementor Pro ;
* un hébergement existant ;
* le nom de domaine principal : https://mfpsport.com/.

Le site est actuellement en production, mais :

* aucun véritable travail SEO n’a été réalisé ;
* le site dépend de WordPress et d’un plugin payant ;
* je souhaite disposer d’un contrôle complet sur le code ;
* je veux supprimer la dépendance à Elementor ;
* je souhaite utiliser des technologies modernes ;
* je ne veux pas ajouter de nouveau coût récurrent.

Le futur site devra donc être réalisé sans WordPress et sans Elementor.

Le projet est disponible dans le dépôt GitHub suivant :

https://github.com/Inzicv/mfpsport

Les anciennes ressources WordPress se trouvent dans le dossier local :

`D:\Git\MFPsport\ressources site`

Ce dossier contient notamment :

* les images provenant de `wp-content` ;
* l’export XML complet du site WordPress ;

Le dossier est déjà lié au dépôt GitHub.

Le site existant doit être étudié comme une source de contenu et de médias, mais il ne faut pas reproduire aveuglément son architecture, son design ou ses défauts.

MFP Sport a besoin d’un **site vitrine**.

Le site ne doit pas gérer directement :

* les comptes clients ;
* les inscriptions ;
* les paiements ;
* les plannings ;
* les disponibilités ;
* les réservations.

Toutes ces fonctionnalités sont gérées par la plateforme externe **Sportigo**.

Le site devra uniquement :

* présenter MFP Sport ;
* expliquer son positionnement ;
* présenter les prestations et les tarifs ;
* présenter les professionnels intervenant dans la salle ;
* publier des actualités ;
* intégrer les outils proposés par Sportigo (Sportigo popose une integration du planning cours collectif en htlm ou iframe);

# 5. Positionnement de MFP Sport

MFP Sport ne doit pas être présenté comme une salle de sport classique ou low cost comparable à Basic-Fit.

La principale différence repose sur un **accompagnement humain, individualisé, premium et pluridisciplinaire**.

MFP Sport s’adresse aussi bien :

* aux sportifs de haut niveau ;
* aux sportifs amateurs ;
* aux équipes professionnelles ou semi-professionnelles ;
* aux personnes souhaitant améliorer leurs performances ;
* aux personnes qui reprennent le sport après une longue interruption ;
* aux personnes ayant un objectif de remise en forme ;
* aux personnes reprenant une activité après un problème de santé ;
* aux personnes qui ont besoin d’un cadre, d’un suivi et d’un programme personnalisé.

Le positionnement doit mettre en avant :

* l’accompagnement personnalisé ;
* l’encadrement par des professionnels ;
* la préparation physique ;
* les bilans corporels et physiques ;
* les outils modernes et technologiques ;
* les infrastructures de qualité ;
* la récupération sportive ;
* la prévention ;
* l’amélioration durable des performances ;
* la remise en forme encadrée ;
* une approche globale du corps et de la progression.

MFP Sport peut proposer un accompagnement pluridisciplinaire avec différents professionnels, notamment :

* coach sportif ;
* préparateur physique ;
* nutritionniste ;
* kinésithérapeute ;
* ostéopathe ;
* masseur ;
* préparateur mental.

Le site doit expliquer clairement que MFP Sport ne propose pas seulement l’accès à une salle et à des machines.

Il propose un environnement dans lequel une personne peut être :

* évaluée ;
* conseillée ;
* accompagnée ;
* entraînée ;
* suivie ;
* aidée dans sa récupération ;
* orientée vers les professionnels adaptés à ses besoins.

# 6. Objectif principal de conversion

L’objectif principal du site est d’inciter le visiteur à réserver une **séance d’essai gratuite**, qui sert également de premier bilan.

Cette séance comprend :

* une séance de sport accompagnée ;
* un échange avec un coach ;
* une première évaluation ;
* la définition des objectifs ;
* l’identification de la formule ou du suivi le plus adapté.

Le site doit construire un parcours de conversion clair :

1. le visiteur comprend immédiatement ce qui différencie MFP Sport ;
2. il reconnaît que l’offre correspond à son profil ou à son besoin ;
3. il découvre les méthodes, les équipements et les professionnels ;
4. il est rassuré sur son niveau et sa capacité à reprendre le sport ;
5. il consulte les prestations ou les formules ;
6. il réserve une séance d’essai gratuite via Sportigo.

Le bouton ou appel à l’action principal doit être cohérent sur tout le site, par exemple :

**Réserver mon bilan gratuit**

Évite les appels à l’action vagues comme « En savoir plus » lorsqu’une formulation plus précise est possible.
